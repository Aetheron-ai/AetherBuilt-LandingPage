# AetherBuilt Landing Page

**Internal Use Only** - AetherBuilt Startup Project

A modern, responsive landing page for AetherBuilt's internal services and solutions.

## Tech Stack

- React 18, TypeScript
- Tailwind CSS, PostCSS
- Vite
- Node.js with Express (server)

## Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/Aetheron-ai/AetherBuilt-LandingPage.git
   cd AetherBuilt-LandingPage
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run development server:
   ```bash
   npm run dev
   ```

## Usage

- Access at [http://localhost:5173](http://localhost:5173)
- Customize sections in `src/components/sections/`
